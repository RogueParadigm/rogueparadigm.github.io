Overviewâ€‹

What is the Targeting System?â€‹

The targeting system allows your player to detect and lock onto enemies for combat:

  • Lock-on targeting (Dark Souls style)
  • Soft targeting (auto-aim assistance)
  • Priority targeting (threat-based)
  • Multi-target management

How It Worksâ€‹

Similar to the interaction system, but typically with:

  • Longer detection range
  • More complex scoring (distance + angle + threat)
  • Target locking mechanism
  • Visual indicators

Basic Setupâ€‹

For a quick setup, see the Quick Start Guide.

Key differences from interaction:

  • Range: Longer (2000-5000 units)
  • Trace Channel: Usually Pawn or Enemy
  • Scoring: Often multi-factor (distance + angle)
  • Interface: IERPTargetable instead of IERPInteractable

  • Lock persists until broken (button press, distance, death)
