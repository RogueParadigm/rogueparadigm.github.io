Default Pipeline Configurationâ€‹

For Interaction:

Channel Id: "Interaction"
Sampler: ERPSphereOverlapSampler (Range: 300, Trace: Visibility)
Filters: ERPInteractableFilter
Scorers: ERPDistanceScorer (Normalize: true)
Aggregator: ERPIdentityAggregator
Resolver: ERPLowestScoreResolver

For Targeting:

Channel Id: "Target"
Sampler: ERPSphereOverlapSampler (Range: 3000, Trace: Pawn)
Filters: ERPTargetableFilter
Scorers: ERPDistanceScorer (Normalize: true)
Aggregator: ERPIdentityAggregator
Resolver: ERPLowestScoreResolver
