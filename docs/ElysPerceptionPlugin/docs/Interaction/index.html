Overviewâ€‹

What is the Interaction System?â€‹

The interaction system allows your player to detect and interact with objects in the world:

  • Doors that can be opened
  • Items that can be picked up
  • NPCs that can be talked to
  • Levers that can be pulled
  • Quest objects that can be examined

How It Worksâ€‹

  1. Detection: The perception system continuously checks for nearby interactable objects
  2. Notification: When an object comes into range, you're notified via events
  3. UI Update: You show a prompt to the player (e.g., "Press E to open")
  4. Execution: When the player presses the button, you execute the interaction
  5. Response: The object responds (door opens, item is picked up, etc.)

Key Componentsâ€‹

  • ERPPerceptionComponent: Detects interactable objects
  • ERPChannelListenerComponent: Listens for the closest interactable
  • IERPInteractable: Interface that objects implement to be interactable
  • ERPOutlineComponent (optional): Visual highlight for interactable objects

Basic Setupâ€‹

This is covered in the Quick Start Guide, but here's a quick recap:

Player Setupâ€‹

  1. Add ERPPerceptionComponent to your PlayerController/Pawn
  2. Add ERPChannelListenerComponent with Channel Id = Interaction
  3. Configure the Interaction Pipeline with short range (300-500 units)

Object Setupâ€‹

  1. Add ERPInteractable interface to your actor
  2. Implement Can Be Interacted With By â†’ return true
  3. Add your interaction logic

Multiple Interaction Typesâ€‹

Using Multiple Channelsâ€‹

For complex interactions, use multiple channels:

Interaction Channel: Short range, high priority (pickup items, open doors)
Examine Channel: Medium range, low priority (read signs, inspect objects)

Setup:

  1. Create two pipelines with different ranges
  2. Use two ChannelListenerComponents
  3. Show different UI for each

Example:

  • Near a door: "Press E to Open" (Interaction)
  • Near a statue: "Press F to Examine" (Examine)

Player Controller with Interactionâ€‹

MyPlayerController.h:

#pragma once

#include "CoreMinimal.h"
#include "GameFramework/PlayerController.h"
#include "Core/ERPPerceptionComponent.h"
#include "Core/ERPChannelListenerComponent.h"
#include "MyPlayerController.generated.h"

UCLASS()
class YOURPROJECT_API AMyPlayerController : public APlayerController
{
    GENERATED_BODY()

public:
    AMyPlayerController();

protected:
    virtual void BeginPlay() override;
    virtual void SetupInputComponent() override;

    UPROPERTY(VisibleAnywhere, Category = "Perception")
    UERPPerceptionComponent* PerceptionComponent;

    UPROPERTY(VisibleAnywhere, Category = "Perception")
    UERPChannelListenerComponent* InteractionListener;

private:
    UFUNCTION()
    void OnInteractableAcquired(AActor* Candidate);
    
    UFUNCTION()
    void OnInteractableLost(AActor* Candidate);
    
    void HandleInteract();

    AActor* CurrentInteractable;
};

MyPlayerController.cpp:

#include "MyPlayerController.h"
#include "Interfaces/ERPInteractable.h"

AMyPlayerController::AMyPlayerController()
{
    PerceptionComponent = CreateDefaultSubobject<UERPPerceptionComponent>(TEXT("Perception"));
    InteractionListener = CreateDefaultSubobject<UERPChannelListenerComponent>(TEXT("InteractionListener"));
    InteractionListener->ChannelId = FName("Interaction");
}

void AMyPlayerController::BeginPlay()
{
    Super::BeginPlay();
    
    InteractionListener->OnCandidateAcquired.AddDynamic(this, &AMyPlayerController::OnInteractableAcquired);
    InteractionListener->OnCandidateLost.AddDynamic(this, &AMyPlayerController::OnInteractableLost);
}

void AMyPlayerController::SetupInputComponent()
{
    Super::SetupInputComponent();
    
    InputComponent->BindAction("Interact", IE_Pressed, this, &AMyPlayerController::HandleInteract);
}

void AMyPlayerController::OnInteractableAcquired(AActor* Candidate)
{
    CurrentInteractable = Candidate;
    // Show UI prompt
}

void AMyPlayerController::OnInteractableLost(AActor* Candidate)
{
    CurrentInteractable = nullptr;
    // Hide UI prompt
}

void AMyPlayerController::HandleInteract()
{
    if (CurrentInteractable && CurrentInteractable->Implements<UERPInteractable>())
    {
        IERPInteractable::Execute_OnInteract(CurrentInteractable, this);
    }
}
