Testing Your Setupâ€‹

Debug Visualizationâ€‹

Enable debug drawing to see what the system is detecting:

  1. Select your ERPPerceptionComponent
  2. Enable Debug | Debug Perception
  3. Play the game

You should see:

  • Green sphere: Detection range
  • Red lines: Detected candidates

Common Issuesâ€‹

Nothing is being detected:

  • Check that your detection range is large enough
  • Verify the target actor implements the correct interface
  • Check the trace channel matches your actor's collision
  • Enable debug visualization to see what's happening

Events not firing:

  • Verify you bound to the events in BeginPlay
  • Check that the Channel Id matches on both components
  • Make sure the Can Be Targeted/Interacted With function returns true
