Best Practicesâ€‹

âœ… DOâ€‹

  • Keep descriptors lightweight â€” They're queried frequently
  • Use tags for interaction types â€” More flexible than string matching
  • Return dynamic descriptors â€” State can change between queries
  • Normalize prompt text â€” "Press [E] to [Action]" format
  • Test with multiple interactables nearby â€” Make sure the right one is selected

âŒ DON'Tâ€‹

  • Don't cache descriptors â€” Always pull fresh from the object
