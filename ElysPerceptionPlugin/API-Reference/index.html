API Reference

Complete reference for public classes, interfaces, and events in ElysPerceptionPlugin.

Core Components​

UERPPerceptionComponent​

Main perception component managing multi-channel pipeline execution.

Header: Core/ERPPerceptionComponent.h

Properties​

PropertyTypeDescription
DefaultSamplingRangefloatDefault sampling range (cm) passed to samplers
ChannelPipelinesTArrayMulti-channel pipeline configuration
bProvideViewportInfoWhenAvailableboolInclude viewport size in context (local PC only)
bDebugPerceptionboolEnable debug logging of candidates per channel

Methods​

// Get current best candidate for channel
AActor* GetChannelCandidate(FName ChannelId) const;

// Check if channel has candidate
bool HasChannelCandidate(FName ChannelId) const;

// Get all channel candidates
void GetAllChannelCandidates(TArray<FName>& OutChannelIds, TArray<AActor*>& OutCandidates) const;

Events​

// Channel acquired candidate
UPROPERTY(BlueprintAssignable)
FERPOnChannelCandidateAcquiredSignature OnChannelCandidateAcquired;

// Channel lost candidate
UPROPERTY(BlueprintAssignable)
FERPOnChannelCandidateLostSignature OnChannelCandidateLost;

Event Signatures:

    FERPOnChannelCandidateAcquiredSignature, 
    FName, ChannelId,
    AActor*, Candidate
);

Interfaces​

IERPTargetable​

Opt-in targeting interface for actors.

Header: Core/ERPTargetable.h

UINTERFACE(BlueprintType)
class UERPTargetable : public UInterface { GENERATED_BODY() };

class IERPTargetable
{
    GENERATED_BODY()

public:
    // Return true if this actor can be targeted by TargetingActor
    UFUNCTION(BlueprintNativeEvent, Category = "Perception")
    bool CanBeTargetedBy(AActor* TargetingActor) const;
};

IERPInteractable​

Opt-in interaction interface for actors.

Header: Core/ERPInteractable.h

UINTERFACE(BlueprintType)
class UERPInteractable : public UInterface { GENERATED_BODY() };

class IERPInteractable
{
    GENERATED_BODY()

public:
    // Return true if can be interacted with
    UFUNCTION(BlueprintNativeEvent, Category = "Interaction")
    bool CanBeInteractedWith(AActor* InteractingActor) const;

    // Provide interaction descriptor
    UFUNCTION(BlueprintNativeEvent, Category = "Interaction")
    void GetInteractionDescriptor(FERPInteractionDescriptor& OutDescriptor) const;

    // Execute interaction
    UFUNCTION(BlueprintNativeEvent, Category = "Interaction")
    void Interact(AActor* InteractingActor);
};

Default Implementations​

Samplers​

UERPSphereOverlapSampler - Sphere overlap query around context origin

Properties:

  • TraceChannel - Collision channel for query
  • ActorClassFilter - Optional actor class filter
  • bIgnoreContextActor - Ignore the perception owner

Filters​

UERPTargetableFilter - Requires IERPTargetable interface

UERPInteractableFilter - Requires IERPInteractable interface

UERPDistanceFilter - Range-based filtering

Properties:

  • Distance - Threshold radius (cm)
  • PassIfCloser - Keep closer (true) or farther (false)

Scorers​

UERPDistanceScorer - Scores based on distance

Properties:

  • bNormalizeByRange - Normalize score by sampling range
  • bRejectBeyondMaxDistance - Reject candidates beyond range

UERPConeScorer - Scores based on angle from forward direction

Properties:

  • ConeHalfAngleDegrees - Cone aperture (default: 45)
  • bRejectOutsideCone - Reject candidates outside cone

UERPViewportEllipseScorer - Scores based on screen position

Properties:

  • EllipseRadiusPx - Ellipse radius in pixels
  • bRejectOutsideEllipse - Reject candidates outside ellipse

Aggregators​

UERPIdentityAggregator - Returns first score (for single scorer)

UERPAdditiveAggregator - Sums all scores

UERPMultiplicativeAggregator - Multiplies all scores

Resolvers​

UERPLowestScoreResolver - Selects candidate with lowest score

UERPHighestScoreResolver - Selects candidate with highest score

Data Structures​

FERPPerceptionContext​

Immutable snapshot of perception state.

struct FERPPerceptionContext
{
    AActor* ContextActor;        // Perception owner
    FVector Origin;              // Owner position
    FVector Forward;             // Facing direction
    
    // Optional aim ray
    bool bHasAimRay;
    FVector AimOrigin;
    FVector AimDirection;
    
    // Optional viewport projection
    bool bHasViewportProjection;
    FVector2D ViewportCenterPx;
    float ViewportEllipseHalfWidthPx;
    float ViewportEllipseHalfHeightPx;
};

FERPInteractionDescriptor​

Interaction metadata pulled from interactable actors.

struct FERPInteractionDescriptor
{
    FString DisplayName;          // Actor name for UI
    FString PromptText;           // "Press E to Open"
    TArray<FName> InteractionTags; // Interaction types
    FString PresentationHints;    // Optional rendering hints
};

FERPChannelPipeline​

Pipeline configuration for one channel.

struct FERPChannelPipeline
{
    FName ChannelId;                           // Channel identifier
    UERPSamplerBase* Sampler;                  // Required
    TArray<UERPFilterBase*> Filters;           // Optional
    TArray<UERPScorerBase*> Scorers;           // Optional
    UERPAggregatorBase* Aggregator;            // Optional
    UERPResolverBase* Resolver;                // Required
};
