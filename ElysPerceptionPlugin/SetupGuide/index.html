Setup Guide

Complete installation and configuration guide for ElysPerceptionPlugin.

Prerequisites​

  • Unreal Engine 5.0 or later
  • Basic understanding of Unreal Engine components and actors

Installation​

1. Copy the Plugin​

Copy ElysPerceptionPlugin folder into your project's Plugins directory:

YourProject/
└── Plugins/
    └── ElysPerceptionPlugin/

2. Enable the Plugin​

  1. Open your project in Unreal Editor
  2. Go to Edit → Plugins
  3. Search for "Elys Perception Plugin"
  4. Check the box to enable it
  5. Restart the editor when prompted

3. Verify Installation​

In the Content Browser, enable "Show Plugin Content" and verify the ElysPerceptionPlugin folder appears.

4. Update Build Configuration (C++ Projects)​

Add the module to your .Build.cs file:

PublicDependencyModuleNames.AddRange(new string[] { 
    "Core", 
    "CoreUObject", 
    "Engine", 
    "ElysPerceptionPlugin"  // Add this
});

Quick Start​

Blueprint Setup​

1. Add Perception Component​

Select your PlayerController or Pawn and add ERPPerceptionComponent.

2. Configure Target Pipeline​

In component details, expand Perception | Pipelines → Target Pipeline:

Basic Settings:

  • Channel Id: Target
  • Default Sampling Range: 2000.0

Pipeline Stages:

  1. Sampler (click + to add):

    • Type: ERPSphereOverlapSampler
    • Trace Channel: Visibility
    • Ignore Context Actor: ✓

  2. Filters:

    • Type: ERPTargetableFilter

  3. Scorers:

    • Type: ERPDistanceScorer
    • Normalize By Range: ✓

  4. Aggregator:

    • Type: ERPIdentityAggregator

  5. Resolver:

    • Type: ERPLowestScoreResolver

3. Make Actors Targetable​

On target actors (enemies, NPCs):

  1. Class Settings → Interfaces → Add ERPTargetable
  2. In Event Graph, implement Can Be Targeted By
  3. Return true (or add custom logic)

4. Add Channel Listener​

On your PlayerController or Pawn:

  1. Add Component → ERPChannelListenerComponent
  2. Set Channel Id to Target
  3. Bind to events:
Event BeginPlay
  → Get Component (TargetListener)
  → Bind Event to OnCandidateAcquired
    → Print String: "Target Acquired: {Candidate}"

Blueprint Example​

Complete Target System​

Variables:

  • CurrentTarget (Actor reference)

Event Graph:

// Handle target acquisition
Event: OnCandidateAcquired (Candidate)
  → Set CurrentTarget = Candidate
  → Print: "Target Acquired"
  → (Update UI widget to show target indicator)

// Handle target lost
Event: OnCandidateLost (Candidate)
  → Set CurrentTarget = None
  → Print: "Target Lost"
  → (Remove UI indicator)

// Get current target on demand
Event Input: AttackButton
  → TargetListener → GetCurrentCandidate
  → IsValid?
    True: → Execute attack on target
    False: → Print: "No target available"

C++ Setup​

1. Create Component​

YourPlayerController.h:

#pragma once

#include "CoreMinimal.h"
#include "GameFramework/PlayerController.h"
#include "Core/ERPPerceptionComponent.h"
#include "Core/ERPChannelListenerComponent.h"
#include "YourPlayerController.generated.h"

UCLASS()
class YOURPROJECT_API AYourPlayerController : public APlayerController
{
    GENERATED_BODY()

public:
    AYourPlayerController();

protected:
    virtual void BeginPlay() override;

    UPROPERTY(VisibleAnywhere, BlueprintReadOnly, Category = "Perception")
    UERPPerceptionComponent* PerceptionComponent;

    UPROPERTY(VisibleAnywhere, BlueprintReadOnly, Category = "Perception")
    UERPChannelListenerComponent* TargetListener;

private:
    UFUNCTION()
    void OnTargetAcquired(AActor* Candidate);

    UFUNCTION()
    void OnTargetLost(AActor* Candidate);
};

YourPlayerController.cpp:

#include "YourPlayerController.h"
#include "Pipeline/ERPChannelPipeline.h"
#include "Pipeline/Samplers/ERPSphereOverlapSampler.h"
#include "Pipeline/Filters/ERPTargetableFilter.h"
#include "Pipeline/Scorers/ERPDistanceScorer.h"
#include "Pipeline/Aggregators/ERPIdentityAggregator.h"
#include "Pipeline/Resolvers/ERPLowestScoreResolver.h"

AYourPlayerController::AYourPlayerController()
{
    PerceptionComponent = CreateDefaultSubobject<UERPPerceptionComponent>(TEXT("PerceptionComponent"));
    
    TargetListener = CreateDefaultSubobject<UERPChannelListenerComponent>(TEXT("TargetListener"));
    TargetListener->ChannelId = FName("Target");
}

void AYourPlayerController::BeginPlay()
{
    Super::BeginPlay();

    // Configure Target Pipeline
    FERPChannelPipeline& TargetPipeline = PerceptionComponent->TargetPipeline;
    TargetPipeline.ChannelId = FName("Target");

    UERPSphereOverlapSampler* Sampler = NewObject<UERPSphereOverlapSampler>(this);
    Sampler->TraceChannel = ECC_Visibility;
    Sampler->bIgnoreContextActor = true;
    TargetPipeline.Sampler = Sampler;

    UERPTargetableFilter* Filter = NewObject<UERPTargetableFilter>(this);
    TargetPipeline.Filters.Add(Filter);

    UERPDistanceScorer* Scorer = NewObject<UERPDistanceScorer>(this);
    Scorer->bNormalizeByRange = true;
    TargetPipeline.Scorers.Add(Scorer);

    TargetPipeline.Aggregator = NewObject<UERPIdentityAggregator>(this);
    TargetPipeline.Resolver = NewObject<UERPLowestScoreResolver>(this);

    // Bind events
    TargetListener->OnCandidateAcquired.AddDynamic(this, &AYourPlayerController::OnTargetAcquired);
    TargetListener->OnCandidateLost.AddDynamic(this, &AYourPlayerController::OnTargetLost);
}

void AYourPlayerController::OnTargetAcquired(AActor* Candidate)
{
    UE_LOG(LogTemp, Log, TEXT("Target acquired: %s"), *Candidate->GetName());
}

void AYourPlayerController::OnTargetLost(AActor* Candidate)
{
    UE_LOG(LogTemp, Log, TEXT("Target lost: %s"), *Candidate->GetName());
}

2. Implement Targetable Interface​

YourEnemy.h:

#pragma once

#include "CoreMinimal.h"
#include "GameFramework/Character.h"
#include "Core/ERPTargetable.h"
#include "YourEnemy.generated.h"

UCLASS()
class YOURPROJECT_API AYourEnemy : public ACharacter, public IERPTargetable
{
    GENERATED_BODY()

public:
    virtual bool CanBeTargetedBy_Implementation(AActor* TargetingActor) const override;

protected:
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Combat")
    bool bIsAlive = true;
};

YourEnemy.cpp:

#include "YourEnemy.h"

bool AYourEnemy::CanBeTargetedBy_Implementation(AActor* TargetingActor) const
{
    return bIsAlive;
}

Testing​

Verification Checklist​

  • Perception component added to PlayerController/Pawn
  • Channel Listener component added
  • Pipeline configured with all stages
  • Default sampling range set (e.g., 2000)
  • Target actors implement IERPTargetable
  • CanBeTargetedBy returns true
  • OnCandidateAcquired event bound and firing

Debug Visualization​

Enable bDebugPerception on ERPPerceptionComponent to see:

  • Green lines for successful traces
  • Red lines for hits
  • Debug info on screen

Console commands:

showdebug PERCEPTION  // Show perception debug
stat GAME            // Show component tick times

Troubleshooting​

Target Not Detected​

Check Interface: Verify target implements IERPTargetable and CanBeTargetedBy returns true.

Check Range: Increase DefaultSamplingRange or verify actors are within range.

Check Collision: Verify TraceChannel in Sampler matches actor collision settings.

Events Not Firing​

Bind Timing: Bind in BeginPlay, not constructor. Use AddDynamic for delegates.

Check Channel Listener: Verify ChannelId matches pipeline configuration.

Performance Issues​

Reduce Range: Lower DefaultSamplingRange to minimum needed.

Adjust Tick Rate: Use PrimaryComponentTick.TickInterval if real-time isn't needed.

Optimize Sampler: Use tighter Actor Class filter.
