?? Usage Example (Minimal Setup)‚Äã

Interactable Actor‚Äã

Components:
  - StaticMeshComponent
  - ERPBaseInteractionFeedbackComponent
    - OutlineComponent: (assign your outline component)
    - WorldSpaceWidgetComponent: (assign your widget component)

Interfaces:
  - IERPInteractable

PlayerController‚Äã

Event BeginPlay:
  ? Get Component: ERPInteractionComponent
  ? Bind to OnCandidateChanged: HandleCandidateChanged
  ? Bind to OnDescriptorChanged: HandleDescriptorChanged

Function HandleCandidateChanged(Previous, New, ChannelId):
  ? If Previous valid:
    ? Get Component: ERPBaseInteractionFeedbackComponent (Previous)
    ? SetInteractionFocused(false)
  ? If New valid:
    ? Get Component: ERPBaseInteractionFeedbackComponent (New)
    ? SetInteractionFocused(true)

Function HandleDescriptorChanged(Candidate, Descriptor, ChannelId):
  ? Get Component: ERPBaseInteractionFeedbackComponent (Candidate)
  ? UpdateInteractionDescriptor(Descriptor)

That's it! No custom widgets, no boilerplate, just automatic visual feedback.

Components:

  • ERPInteractionComponent ‚Äî Validates and resolves descriptors

Event Graph Implementation:

OnCandidateChanged Handler:

  • Calls SetInteractionFocused(false) on previous candidate's feedback component
  • Calls SetInteractionFocused(true) on new candidate's feedback component

  • Calls UpdateInteractionDescriptor(Descriptor)

üí° Best Practices‚Äã

‚úÖ DO‚Äã

  • Use ERPInteractionComponent for all interaction state management
  • Add ERPBaseInteractionFeedbackComponent to interactables for quick setup
  • Build descriptors dynamically in GetInteractionDescriptor
  • Always null-check components before calling functions
  • Keep CanBeInteractedWith lightweight (called frequently)

‚ùå DON'T‚Äã

  • Don't put UI logic in PlayerController
  • Don't cache descriptors (always pull fresh)
  • Don't assume feedback component exists (it's optional)
  • Don't put gameplay execution in interaction components
  • Don't create DataAssets for descriptors (use runtime structs)

?? Design Principles Maintained‚Äã
