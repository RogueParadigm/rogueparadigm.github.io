Language Model Guide

Using Large Language Models for NPC dialogue and reasoning in ElysGenAI.

Quick Setup‚Äã

// 1. Add component
UPROPERTY(VisibleAnywhere)
UERP_LLMComponent* LLMComponent;

// 2. Set system prompt
LLMComponent->SetSystemPrompt(TEXT("You are a friendly merchant NPC."));

// 3. Bind event
LLMComponent->OnGenerationComplete.AddDynamic(
    this, &AMyNPC::OnDialogueGenerated);

// 4. Send message
LLMComponent->SendMessage(TEXT("What are you selling?"));

// 5. Handle response
void AMyNPC::OnDialogueGenerated(const FElysLLMResult& Result)
{
    DisplayDialogue(Result.GeneratedText);
}

Models‚Äã

Bundled Model: Phi-3-mini‚Äã

Specs:

  • Size: ~2.7GB (Q4 quantized)
  • Context: 4096 tokens
  • License: MIT
  • Speed: ~20 tokens/sec (CPU)

Use Cases:

  • NPC dialogue
  • Quest generation
  • Item descriptions
  • Dynamic storytelling

Supported Models‚Äã

All GGUF format models from:

  • Phi-3 (Microsoft, MIT) - Bundled
  • Gemma 2 (Google, Apache 2.0)
  • Mistral (Mistral AI, Apache 2.0)
  • Llama 3 (Meta, Llama Community License)
  • TinyLlama (Small, Apache 2.0)

Temperature & Sampling‚Äã

Temperature‚Äã

Controls randomness:

// Conservative (factual, consistent)
LLMComponent->SetTemperature(0.3f);

// Balanced (default)
LLMComponent->SetTemperature(0.7f);

// Creative (varied, unpredictable)
LLMComponent->SetTemperature(1.5f);

Guidelines:

  • 0.0-0.3: Quest dialogs, tutorials
  • 0.5-0.8: NPC conversations
  • 1.0-2.0: Creative writing, variety

Examples‚Äã

NPC Merchant‚Äã

UCLASS()
class AMerchantNPC : public ACharacter
{
    UPROPERTY(VisibleAnywhere)
    UERP_LLMComponent* LLMComponent;
    
    void BeginPlay() override
    {
        Super::BeginPlay();
        
        LLMComponent->SetSystemPrompt(
            TEXT("You are Gareth, a friendly blacksmith. "
                 "You sell weapons and armor. "
                 "You're jovial and love to chat about crafting."));
        
        LLMComponent->SetTemperature(0.8f);
        LLMComponent->SetMaxTokens(100);
    }
    
    UFUNCTION(BlueprintCallable)
    void TalkTo(const FString& PlayerMessage)
    {
        LLMComponent->SendMessage(PlayerMessage);
    }
    
    UFUNCTION()
    void OnDialogueGenerated(const FElysLLMResult& Result)
    {
        DisplayDialogueBubble(Result.GeneratedText);
        PlayVoiceSound();
    }
};

Quest Generator‚Äã

FString GenerateQuest(const FString& QuestType, const FString& Location)
{
    FString Prompt = FString::Printf(
        TEXT("Generate a %s quest that takes place in %s. "
             "Include: objective, reward, and a twist. "
             "Keep it under 100 words."),
        *QuestType, *Location);
    
    LLMComponent->SendMessageSync(Prompt);  // Blocking call
    return LLMComponent->GetLastResponse();
}

Dynamic Item Descriptions‚Äã

void GenerateItemLore(UInventoryItemERP* Item)
{
    FString Prompt = FString::Printf(
        TEXT("Write a brief fantasy lore description for: %s. "
             "Make it mysterious and intriguing. Max 50 words."),
        *Item->ItemName);
    
    LLMComponent->OnGenerationComplete.AddDynamic(
        this, &UInventorySystem::OnLoreGenerated);
    
    LLMComponent->SendMessage(Prompt);
}

Streaming Responses‚Äã

Token-by-Token‚Äã

void AMerchantNPC::BeginPlay()
{
    Super::BeginPlay();
    
    // Enable streaming
    LLMComponent->SetStreamingEnabled(true);
    
    // Bind token event
    LLMComponent->OnTokenGenerated.AddDynamic(
        this, &AMerchantNPC::OnTokenReceived);
}

void AMerchantNPC::OnTokenReceived(const FString& Token)
{
    // Append to dialogue (typewriter effect)
    CurrentDialogue += Token;
    UpdateDialogueWidget(CurrentDialogue);
}

void AMerchantNPC::OnDialogueComplete(const FElysLLMResult& Result)
{
    // Final cleanup
    FinalizeDialogue(Result.GeneratedText);
}

Performance‚Äã

Generation Speed‚Äã

Phi-3-mini (Q4):

  • CPU (8-core): ~15-25 tokens/sec
  • CPU (16-core): ~30-50 tokens/sec
  • GPU (future): ~100+ tokens/sec

Latency:

  • First token: ~200-500ms
  • Subsequent tokens: ~40-70ms each

Memory Usage‚Äã

  • Phi-3-mini Q4: ~2.7GB RAM
  • Mistral 7B Q4: ~4GB RAM
  • Llama 3 8B Q4: ~4.5GB RAM

Optimization‚Äã

// Reduce context window for faster inference
Settings->ContextLength = 2048;  // Default: 4096

// Use fewer threads if CPU-bound
Settings->NumThreads = 2;

// Shorter responses
LLMComponent->SetMaxTokens(50);  // Default: 256

Troubleshooting‚Äã

Slow Generation‚Äã

  1. Reduce MaxTokens
  2. Use smaller model (Phi-3 ‚Üí TinyLlama)
  3. Reduce NumThreads
  4. Close other CPU-intensive apps

Out of Memory‚Äã

  1. Use smaller model
  2. Reduce ContextLength
  3. Clear history more frequently
  4. Close other memory-intensive apps

Inconsistent Responses‚Äã

  1. Lower Temperature (0.7 ‚Üí 0.3)
  2. Use more specific system prompts
  3. Provide examples in prompt

Model Not Loading‚Äã

  1. Check model path in settings
  2. Verify GGUF format
  3. Check file permissions
  4. Check Output Log for errors
