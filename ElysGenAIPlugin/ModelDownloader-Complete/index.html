Whisper Model Downloader - Dual Editor/Runtime System

The ElysGenAIPlugin includes a flexible model downloader that works both in the editor and in packaged games.

üì¶ Two Systems, One Codebase‚Äã

üîß Editor Downloads (Development Time)‚Äã

Use UERP_ModelDownloaderEditorSubsystem for downloading models during development.

üéÆ Runtime Downloads (Packaged Games)‚Äã

Use UERP_ModelDownloaderSubsystem for allowing players to download models in packaged games.

üîß Editor Usage‚Äã

Method 1: Menu (Easiest)‚Äã

  1. Open Unreal Editor
  2. Go to Tools ‚Üí Elys GenAI ‚Üí Download AI Models
  3. Follow the prompts to download a model

Method 2: Blueprint (Editor Utility Widget)‚Äã

Create an Editor Utility Widget and use these nodes:

Get Editor Subsystem (UERP_ModelDownloaderEditorSubsystem)
‚Üí Download Whisper Model (ModelId: "base")

Bind to events for progress:

Get Editor Subsystem
‚Üí Get Model Downloader
‚Üí Bind Event to OnDownloadProgress
‚Üí Bind Event to OnDownloadComplete

Method 3: C++ (Editor Code)‚Äã

#include "ERP_ModelDownloaderEditorSubsystem.h"

// Get the subsystem
auto* Downloader = GEditor->GetEditorSubsystem<UERP_ModelDownloaderEditorSubsystem>();

// Download a model
if (Downloader)
{
    Downloader->DownloadWhisperModel(TEXT("base"));
}

Method 4: Output Log Command‚Äã

Open Window ‚Üí Developer Tools ‚Üí Output Log and paste:

auto* Subsystem = GEditor->GetEditorSubsystem<UERP_ModelDownloaderEditorSubsystem>();
Subsystem->DownloadWhisperModel(TEXT("base"));

üéÆ Runtime Usage (Packaged Games)‚Äã

Blueprint Setup‚Äã

  1. Create a UI widget for players
  2. Get the Runtime Subsystem:
Get Game Instance
‚Üí Get Subsystem (UERP_ModelDownloaderSubsystem)
  1. Download on button press:
Button Clicked
‚Üí Get Subsystem (UERP_ModelDownloaderSubsystem)
‚Üí Download Whisper Model (ModelId: "base")
  1. Show progress:
Get Subsystem
‚Üí Get Model Downloader
‚Üí Bind Event to OnDownloadProgress (update progress bar)
‚Üí Bind Event to OnDownloadComplete (show completion message)

C++ Runtime‚Äã

#include "ERP_ModelDownloaderSubsystem.h"

// Get the subsystem
auto* Downloader = GetGameInstance()->GetSubsystem<UERP_ModelDownloaderSubsystem>();

// Download a model
if (Downloader)
{
    // Bind to events
    auto* ModelDownloader = Downloader->GetModelDownloader();
    ModelDownloader->OnDownloadProgress.AddDynamic(this, &UMyWidget::OnProgress);
    ModelDownloader->OnDownloadComplete.AddDynamic(this, &UMyWidget::OnComplete);
    
    // Start download
    Downloader->DownloadWhisperModel(TEXT("base"));
}

Example: In-Game Download Screen‚Äã

UCLASS()
class UMyDownloadWidget : public UUserWidget
{
    GENERATED_BODY()

public:
    UPROPERTY(meta = (BindWidget))
    UProgressBar* DownloadProgress;
    
    UPROPERTY(meta = (BindWidget))
    UTextBlock* StatusText;
    
    UPROPERTY(meta = (BindWidget))
    UButton* DownloadButton;

protected:
    virtual void NativeConstruct() override
    {
        Super::NativeConstruct();
        
        DownloadButton->OnClicked.AddDynamic(this, &UMyDownloadWidget::OnDownloadClicked);
    }
    
    UFUNCTION()
    void OnDownloadClicked()
    {
        auto* Downloader = GetGameInstance()->GetSubsystem<UERP_ModelDownloaderSubsystem>();
        if (Downloader)
        {
            auto* ModelDownloader = Downloader->GetModelDownloader();
            ModelDownloader->OnDownloadProgress.AddDynamic(this, &UMyDownloadWidget::OnProgress);
            ModelDownloader->OnDownloadComplete.AddDynamic(this, &UMyDownloadWidget::OnComplete);
            
            Downloader->DownloadWhisperModel(TEXT("base"));
            StatusText->SetText(FText::FromString(TEXT("Downloading...")));
        }
    }
    
    UFUNCTION()
    void OnProgress(const FString& ModelId, int64 BytesDownloaded, int64 TotalBytes, float Percentage)
    {
        DownloadProgress->SetPercent(Percentage / 100.0f);
        StatusText->SetText(FText::FromString(FString::Printf(
            TEXT("Downloading: %.1f%%"), Percentage
        )));
    }
    
    UFUNCTION()
    void OnComplete(const FString& ModelId, bool bSuccess, const FString& FilePath, const FString& Error)
    {
        if (bSuccess)
        {
            StatusText->SetText(FText::FromString(TEXT("Download Complete!")));
        }
        else
        {
            StatusText->SetText(FText::FromString(FString::Printf(
                TEXT("Download Failed: %s"), *Error
            )));
        }
    }
};

üíæ Model Storage‚Äã

Editor (Development)‚Äã

<ProjectDirectory>/Content/ElysGenAI/Models/Whisper/

Packaged Game (Runtime)‚Äã

<GameInstallDirectory>/Content/ElysGenAI/Models/Whisper/

Models are stored in the same relative path, so they work seamlessly between editor and packaged builds.

‚öôÔ∏è Automatic Configuration‚Äã

Editor‚Äã

When a model download completes in the editor, settings are automatically configured:

  • STTBackend ‚Üí "Whisper"
  • STTModelPath ‚Üí Path to downloaded model
  • bEnableSTT ‚Üí true

Check Project Settings ‚Üí Plugins ‚Üí Elys GenAI Framework to verify.

Runtime‚Äã

For packaged games, you need to set the model path manually after download:

UElysGenAISettings* Settings = GetMutableDefault<UElysGenAISettings>();
Settings->STTModelPath = DownloadedFilePath;
Settings->STTBackend = TEXT("Whisper");
Settings->bEnableSTT = true;

üîç API Reference‚Äã

UERP_ModelDownloaderEditorSubsystem (Editor Only)‚Äã

Functions:

  • GetModelDownloader() - Get downloader instance
  • DownloadWhisperModel(ModelId) - Download a model
  • IsWhisperModelDownloaded(ModelId) - Check if exists
  • GetAvailableWhisperModels() - List all models
  • OpenModelsFolder() - Open folder in explorer
  • ConfigureSettingsForModel(ModelId) - Auto-configure settings

UERP_ModelDownloaderSubsystem (Runtime)‚Äã

Functions: Same as EditorSubsystem, but for packaged games

  • GetModelDownloader()
  • DownloadWhisperModel(ModelId)
  • IsWhisperModelDownloaded(ModelId)
  • GetAvailableWhisperModels()
  • OpenModelsFolder()
  • ConfigureSettingsForModel(ModelId)

UERP_WhisperModelDownloader (Shared Logic)‚Äã

Events:

  • OnDownloadProgress - Progress updates (BytesDownloaded, TotalBytes, Percentage)
  • OnDownloadComplete - Download finished (Success, FilePath, ErrorMessage)

Functions:

  • DownloadModel(ModelId) - Start download
  • CancelDownload() - Cancel current download
  • IsDownloading() - Check if downloading
  • GetDownloadProgress() - Get current progress
  • IsModelDownloaded(ModelId) - Check if model exists
  • GetAvailableModels() - List available models
  • GetModelInfo(ModelId) - Get info about a model

üéØ Best Practices‚Äã

For Developers‚Äã

  1. Download models in editor during development
  2. Test with smaller models first (tiny/base)
  3. Use the editor subsystem for quick iteration

For Players (Packaged Games)‚Äã

  1. Provide clear UI for model download
  2. Show download progress
  3. Allow cancellation
  4. Handle errors gracefully
  5. Recommend base/base.en as default
  6. Make larger models optional downloads
  7. Check available disk space before download
  8. Save model preference to save game

Performance Tips‚Äã

  • Download models on first launch or in main menu (not during gameplay)
  • Cache model availability checks
  • Use async loading for models
  • Consider background downloads with low priority

üö® Troubleshooting‚Äã

Q: Download fails with "HTTP Error"
A: Check internet connection. Models are downloaded from Hugging Face CDN.

Q: Settings not updating in editor
A: Restart the editor after first model download.

Q: Model path wrong in packaged game
A: Ensure Content folder is packaged correctly. Check pak file includes Content/ElysGenAI/.

Q: Can't find subsystem in Blueprint
A: Editor subsystem only works IN EDITOR. Use GameInstance subsystem for runtime.

Q: Download progress not showing
A: Make sure you bind to OnDownloadProgress BEFORE calling DownloadWhisperModel().

üìù Example: Complete In-Game Flow‚Äã

// 1. On game startup, check if model exists
auto* Downloader = GetGameInstance()->GetSubsystem<UERP_ModelDownloaderSubsystem>();
if (!Downloader->IsWhisperModelDownloaded(TEXT("base")))
{
    // Show download prompt to player
    ShowModelDownloadPrompt();
}

// 2. Player clicks download
void ShowDownloadUI()
{
    auto* Downloader = GetGameInstance()->GetSubsystem<UERP_ModelDownloaderSubsystem>();
    auto* ModelDownloader = Downloader->GetModelDownloader();
    
    // Bind events
    ModelDownloader->OnDownloadProgress.AddDynamic(this, &UMyWidget::UpdateProgress);
    ModelDownloader->OnDownloadComplete.AddDynamic(this, &UMyWidget::OnDownloadDone);
    
    // Start download
    Downloader->DownloadWhisperModel(TEXT("base"));
}

// 3. Download completes, configure settings
void OnDownloadDone(const FString& ModelId, bool bSuccess, const FString& Path, const FString& Error)
{
    if (bSuccess)
    {
        // Configure settings
        auto* Downloader = GetGameInstance()->GetSubsystem<UERP_ModelDownloaderSubsystem>();
        Downloader->ConfigureSettingsForModel(ModelId);
        
        // Initialize STT system
        InitializeSTTSystem();
        
        // Continue to main menu
        ShowMainMenu();
    }
}
