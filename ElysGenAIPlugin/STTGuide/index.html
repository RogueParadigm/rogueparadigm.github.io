Speech-to-Text Guide

Complete guide to using Speech-to-Text features in ElysGenAI.

Quick Setup‚Äã

// 1. Add component to Actor
UPROPERTY(VisibleAnywhere, BlueprintReadOnly)
UERP_STTComponent* STTComponent;

// 2. Bind event
STTComponent->OnTranscriptionComplete.AddDynamic(
    this, &AMyActor::OnTranscriptionReceived);

// 3. Start listening
STTComponent->StartListening();

// 4. Handle results
void AMyActor::OnTranscriptionReceived(const FElysSTTResult& Result)
{
    UE_LOG(LogTemp, Log, TEXT("Transcription: %s"), *Result.TranscribedText);
}

Language Support‚Äã

Supported Languages‚Äã

Whisper supports 99+ languages:

// English
SetLanguageCode(TEXT("en"));

// Spanish
SetLanguageCode(TEXT("es"));

// French
SetLanguageCode(TEXT("fr"));

// German
SetLanguageCode(TEXT("de"));

// Japanese
SetLanguageCode(TEXT("ja"));

// Chinese
SetLanguageCode(TEXT("zh"));

Full list: https://github.com/openai/whisper#available-models-and-languages

English-Only Models‚Äã

Models ending in .en are English-only but faster:

  • tiny.en, base.en, small.en, medium.en

Use multilingual models for other languages:

  • tiny, base, small, medium, large-v3

Voice Activity Detection (VAD)‚Äã

VAD automatically detects speech and filters silence:

// Enable VAD (recommended)
STTComponent->SetEnableVAD(true);

// VAD sensitivity (Project Settings)
Settings->VADThreshold = 0.5f;  // 0.0 (sensitive) - 1.0 (strict)

Benefits:

  • Reduces false transcriptions
  • Saves CPU (only processes speech)
  • Improves accuracy

Examples‚Äã

Command Recognition‚Äã

void AMyActor::OnTranscriptionReceived(const FElysSTTResult& Result)
{
    if (Result.Confidence < 0.7f) return;  // Ignore low confidence
    
    FString Command = Result.TranscribedText.ToLower();
    
    if (Command.Contains(TEXT("attack")))
    {
        ExecuteAttack();
    }
    else if (Command.Contains(TEXT("defend")))
    {
        ExecuteDefend();
    }
    else if (Command.Contains(TEXT("inventory")))
    {
        OpenInventory();
    }
}

Dictation Mode‚Äã

UCLASS()
class UDictationComponentERP : public UActorComponent
{
    GENERATED_BODY()
    
    UPROPERTY(VisibleAnywhere)
    UERP_STTComponent* STTComponent;
    
    FString DictatedText;
    
    UFUNCTION()
    void OnTranscription(const FElysSTTResult& Result)
    {
        // Append to running dictation
        if (!DictatedText.IsEmpty())
        {
            DictatedText += TEXT(" ");
        }
        DictatedText += Result.TranscribedText;
        
        OnDictationUpdated.Broadcast(DictatedText);
    }
    
    UPROPERTY(BlueprintAssignable)
    FOnDictationUpdated OnDictationUpdated;
};

Keyword Spotting‚Äã

void AMyActor::OnTranscriptionReceived(const FElysSTTResult& Result)
{
    static const TArray<FString> Keywords = {
        TEXT("help"),
        TEXT("emergency"),
        TEXT("medic"),
        TEXT("enemy")
    };
    
    for (const FString& Keyword : Keywords)
    {
        if (Result.TranscribedText.Contains(Keyword, ESearchCase::IgnoreCase))
        {
            OnKeywordDetected.Broadcast(Keyword);
            break;
        }
    }
}

Performance Optimization‚Äã

Thread Count‚Äã

// Project Settings ‚Üí STT ‚Üí Num Threads
Settings->NumThreads = 4;  // Match CPU core count

Guidelines:

  • 4 cores: 2-3 threads
  • 8 cores: 4-6 threads
  • 16 cores: 6-8 threads

Buffer Size‚Äã

// Larger buffer = more audio context = better accuracy
// But also = more latency
Settings->STTBufferDuration = 3000;  // 3 seconds (default)

Model Selection‚Äã

For real-time games, use base.en:

  • Fast enough for 30-60 FPS
  • Good accuracy
  • Small memory footprint

Troubleshooting‚Äã

No Transcriptions‚Äã

  1. Check microphone permissions
  2. Verify IsListening() returns true
  3. Check Output Log for errors: 
    LogElysSTT: Error: Model not loaded
  4. Speak louder/clearer
  5. Enable VAD

Low Confidence Scores‚Äã

Solutions:

  • Reduce background noise
  • Speak closer to microphone
  • Use better microphone quality
  • Use larger model (base ‚Üí small)
  • Adjust VAD threshold

High Latency‚Äã

Causes:

  • Model too large
  • Too many inference threads
  • CPU bottleneck

Solutions:

  • Use smaller model (small ‚Üí base)
  • Reduce thread count
  • Lower buffer duration

Wrong Language Detected‚Äã

Solution:

  • Set language code explicitly: 
    STTComponent->SetLanguageCode(TEXT("en"));
  • Use language-specific model (base.en)
