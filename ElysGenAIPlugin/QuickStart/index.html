Advanced: C++ Versionâ€‹

Prefer C++? Here's the equivalent code:

// In your PlayerController.h
UPROPERTY(VisibleAnywhere, BlueprintReadOnly)
UERP_STTComponent* STTComponent;

UFUNCTION()
void OnTranscriptionReceived(const FElysSTTResult& Result);

// In your PlayerController.cpp
void AMyPlayerController::BeginPlay()
{
    Super::BeginPlay();
    
    // Create component
    STTComponent = NewObject<UERP_STTComponent>(this);
    STTComponent->RegisterComponent();
    STTComponent->SetLanguageCode(TEXT("en"));
    STTComponent->SetAutoStartListening(true);
    
    // Bind event
    STTComponent->OnTranscriptionComplete.AddDynamic(
        this, &AMyPlayerController::OnTranscriptionReceived
    );
    
    STTComponent->StartListening();
}

void AMyPlayerController::OnTranscriptionReceived(const FElysSTTResult& Result)
{
    if (Result.TranscribedText.Contains(TEXT("hello")))
    {
        GEngine->AddOnScreenDebugMessage(-1, 5.0f, FColor::Green, 
            TEXT("Hello, I heard you!"));
    }
}

Troubleshootingâ€‹

No Transcription Appearingâ€‹

Check:

  1. Microphone permissions granted (Windows: Privacy Settings)
  2. Auto Start Listening is checked
  3. Speak clearly and loudly
  4. Check Output Log for errors

Low Confidence Scoresâ€‹

Solutions:

  1. Reduce background noise
  2. Speak closer to microphone
  3. Use push-to-talk (see Audio Capture Guide)
  4. Adjust VAD threshold in settings

"Model Not Found" Errorâ€‹

Solution:

  1. Verify Whisper model exists at: ElysGenAIPlugin/Content/AI/Models/Whisper/whisper-base.en.bin
  2. Check Project Settings â†’ Elys GenAI â†’ STT Backend â†’ Model Path
